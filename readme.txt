Enables subscribable.com users to subscribe to your blog.

== Description ==

This plugin enables [subscribable.com](http://subscribable.com) users to subscribe to your blog and receive immediate updates each time you publish a new post. It also adds a menu item to the WordPress administration dashboard that you can use to manage your subscribers and a sidepanel widget so that you can display them on your blog. It works with WordPress classic but has not been tested on WordPress MU.

== Installation ==

If you have ever installed a plugin, this should be pretty easy.

1. Download the archive and extract the files. Copy the `subscribable` directory into `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu item on the WordPress administration dashboard
3. Customize the plugin options to your liking through the 'Subscribable' menu item on the WordPress administration dashboard
